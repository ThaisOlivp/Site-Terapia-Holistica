## ✦ About the Project

This website was designed and developed from scratch for a real client — a holistic therapist and spiritual practitioner who serves clients across Brazil and 7+ countries online. The goal was to create a digital presence that matched her brand identity: elegant, spiritual, and trustworthy.

The site functions as both a **service showcase** and a **conversion tool**, guiding visitors from awareness to booking entirely through WhatsApp — no third-party booking system required.

---

## Features

**Homepage**
- Hero section with animated background and brand identity
- "When to seek therapy" section with symptom cards
- Areas of transformation grid organized by life category
- Stats banner: 22,000+ sessions · 7+ countries · 100% online
- Client testimonials with real reviews
- WhatsApp floating CTA button

**Service Pages**
- 8 individual service pages (Tarot, Jogo de Búzios, Mesa Radiônica, Limpeza Energética, Trabalhos Espirituais, Runas das Bruxas, Apometria, Reiki)
- Each page includes a step-by-step booking flow, modality options, and pricing
- Dual pricing in BRL and EUR for international clients
- Direct WhatsApp booking buttons per service

**Shop**
- Spiritual items with product photos, descriptions, and dual-currency pricing
- Individual product detail pages
- Purchase flow via WhatsApp

**UX Details**
- FAQ accordion with smooth expand/collapse
- Sticky navigation with smooth scroll to sections
- Back-to-top button
- "Back to site" navigation on subpages
- Responsive layout for mobile and desktop

## Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Semantic page structure across multiple pages |
| CSS3 | Custom dark theme, animations, responsive grid |
| JavaScript | FAQ accordion, scroll behavior, dynamic interactions |
| Google Fonts | Cinzel (headings) + Cormorant Garamond (body) |
| CSS Custom Properties | Centralized gold/purple color system |

No frameworks, no dependencies, no build tools — pure HTML, CSS and JavaScript.

---

## Design Decisions

- **Dark theme** — chosen to evoke mystery and spirituality while keeping the content readable
- **Gold + purple palette** — reflects luxury, spirituality, and the brand's visual identity
- **Cinzel typeface** — classic serif with small-caps feel, reinforces the mystical and ceremonial tone
- **WhatsApp-first booking** — client's preferred workflow; avoids friction of forms or third-party schedulers
- **Dual currency (BRL/EUR)** — client serves both Brazilian and European clients, pricing clarity reduces barriers
