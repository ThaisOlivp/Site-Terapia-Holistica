# Vivian Pimentel | Holistic Therapy Website

A fully responsive personal website built for a holistic therapist based in Portugal, offering spiritual and energetic healing services online to clients across Brazil, Portugal, and 7+ countries.

## 🌿 About the Project

This project was developed as a professional portfolio website to present services, attract clients, and enable direct booking through WhatsApp. The design reflects the spiritual and mystical nature of the work, using a deep navy palette with gold and lavender accents.

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Smooth scroll animations on page load and scroll
- Sticky navigation bar with hover effects
- 8 individual service pages with detailed descriptions and pricing
- 3 product pages with WhatsApp purchase integration
- Testimonials section
- Client statistics counter
- Privacy Policy page (GDPR compliant — Portuguese law)
- Back-to-top button
- Social media links (WhatsApp, Instagram, Facebook, Email)

## 🛠️ Technologies Used

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript
- Google Fonts (Cinzel + Raleway)
- Font Awesome 4.7
